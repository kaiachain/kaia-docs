# 管理校验器信息

入职完成后，您就可以更新验证器的信息了。

![带有内置验证器的主页](/img/nodes/become-a-validator/image31.png)

以验证器管理器的身份登录并导航到 \*\* 主页\*\* 菜单后，您就可以看到板载的验证器了。

在此页面中，点击 **[管理验证程序]**，进入验证程序管理页面。

管理验证器屏幕](/img/nodes/become-a-validator/image32.png)

在这里您可以执行以下操作。

1. \*\* 传输管理器\*\* - 更改验证管理器帐户。

2. **请求离线** - 提交离线请求以移除验证器。

3. **添加定标合同** - 添加**辅助定标合同**。 您可以操作多个注资合同（**多重注资合同**），以实现分离注资等目的。 请注意，无权限第二阶段**将不支持此功能，因此我们**强烈建议不要使用\*\*，除非您有特殊原因。

4. **删除定标合同** - 删除辅助定标合同。 这将删除已注册到验证器的多个定投合约中的一个。 请注意，不能移除用于入会的主注资合约（**共识注资合约**）；只能移除后来添加的辅助注资合约。
