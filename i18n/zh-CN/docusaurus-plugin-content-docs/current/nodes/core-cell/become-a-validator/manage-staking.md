# 管理定标合同

![管理定标菜单](/img/nodes/become-a-validator/image23.png)

在**管理定标**菜单中，您可以修改先前部署的定标合同的信息。

如果您使用部署了合约的钱包进行连接，您可以浏览定投合约列表，选择特定合约，然后点击 **[管理定投]**，进入其管理界面。

![输入预订合同地址](/img/nodes/become-a-validator/image24.png)

或者，如果您使用任何注资合约管理钱包登录，您可以在 **[输入 CnStaking 合约地址以进行管理]** 框中输入注资合约地址，进入管理界面。

![定标管理概览](/img/nodes/become-a-validator/image25.png)

在该屏幕上，您可以执行以下操作。

## 行政管理<a id="admin-management"></a>

1. **添加管理员** - 添加定标合同管理员。
2. **删除管理员** - 删除定桩合同管理员。
3. **更新要求** - 更改定投合约的多位数阈值。

## 定标操作<a id="staking-operations"></a>

注资/取消注资/待提款](/img/nodes/become-a-validator/image26.png)
![注资操作详情](/img/nodes/become-a-validator/image27.png)

4. \*\* 注 KAIA\*\*--如果公共代表团被禁用，您可以自行决定注 KAIA。
5. \*\* 解除对 KAIA 的认股权\*\* - 如果公共委托被禁用，认股合同管理员可自行决定解除对 KAIA 的认股权。
6. **等待提款** - 点击取消提款按钮后，您可在**7天**至**14天**后申请 KAIA。

## 地址和跟踪器管理<a id="address-and-tracker-management"></a>

![奖励地址和投票人](/img/nodes/become-a-validator/image28.png)

7. **奖励地址** - 如果已禁用公共授权，则可以更改奖励地址。
8. \*\* 选民和定标跟踪器\*\* - 更改用于 GC 投票的选民地址。 更新 Staking Tracker 时，您还可以更改 Staking Tracker 地址。

## 转授权<a id="redelegation"></a>

![Redelegation](/img/nodes/become-a-validator/image29.png)

9. **重新授权** - 启用或禁用重新授权功能，该功能可免除 7 天的解除委托期，并立即在委托合同之间移动 KAIA。 来源地和目的地的盯人合约都必须启用再委托功能。 详见 [KIP-163](https://kips.kaia.io/KIPs/kip-163)。

## 多 ID 申请<a id="multisig-requests"></a>

![Multisig请求](/img/nodes/become-a-validator/image30.png)

10. \*\* 多重登录请求\*\* - 当多重登录阈值为 2 或更高时，执行 1-9 项中的任何操作都需要得到定桩合同管理员的确认。 对于尚未获得足够确认的申请，每个管理员都可以通过该界面登录并执行确认流程。 一旦收集到足够的确认信息，就会执行变更请求。
