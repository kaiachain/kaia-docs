# トランザクション・ビルダーを使用する

:::caution 日没に関するお知らせ

`safe.kaia.io` は **2026年8月9日** にサービス終了となります。 今後は、[app.safe.global](https://app.safe.global) の「Safe Wallet for Kaia Network」をご利用いただき、アカウントの管理を行ってください。 現在お持ちの「Safe Accounts」は、「Safe Wallet」と自動的に互換性が確保されます。

:::

**トランザクション・ビルダー**は、複数の操作を1つのSafeトランザクションにまとめて処理するSafeアプリです。 転送や契約の呼び出しを1つずつ確認するのではなく、バッチを作成し、それをまとめて確認・実行します。

安全なアプリの提供状況は、ネットワークやカタログによって異なる場合があります。 Safe Walletで\*\*「アプリ」**を開き、**「Transaction Builder」\*\*を検索して、KaiaまたはKairos Safe用に起動してください。

最新のUIに対応した製品のヘルプについては、[Safe Wallet ヘルプセンター](https://help.safe.global)をご覧ください。

## KAIAトークンの送金<a id="token-transfer"></a>

**手順 1：** Safe Wallet で **アプリ** を開き、**トランザクションビルダー** を起動します。

**手順 2：** 受信者のメールアドレスを入力してください。 単純なKAIA転送の場合は、ABIを空欄のままにしておいてください。

**手順 3：** 送信する KAIA の値を入力し（例：1 KAIA の場合は `1`）、**取引を追加** をクリックします。

**手順 4：** 一括処理の対象となる各受信者について、この操作を繰り返します。

**ステップ 5：** バッチの作成が完了したら、\*\*「バッチを作成」**をクリックし、処理内容を確認してから、**「バッチを送信」\*\*をクリックし、他の Safe 取引と同様の方法で必要な Safe 署名を集めます。

## 契約上のやり取り<a id="contract-interactions"></a>

1つのSafeトランザクション内で、類似した契約呼び出し（例えば、同じトークンを複数のアドレスに転送するなど）を多数行う必要がある場合は、トランザクションビルダーを使用してください。

**手順 1：** 「Safe Apps」から **Transaction Builder** を開きます。

**手順 2：** **トークン（またはコントラクト）アドレス** と **ABI** を入力します。

**ステップ 3：** メソッド（例：`transfer`）を選択し、パラメータを入力します。

> 注：整数の金額は通常、トークンの最小単位で表記されます（フィールドには小数点以下は含まれません）。 18桁のトークンの場合、`10`というトークンは、しばしば`10000000000000000000`と入力されます。

**手順 4：** **「取引を追加」** をクリックし、各通話についてこの操作を繰り返した後、**「バッチを作成」** → **「バッチを送信」** の順に操作し、Safeでの確認を完了してください。

バッチの契約および譲渡は慎重に行ってください。署名するすべての所有者は、実行前にバッチ全体を確認する必要があります。
