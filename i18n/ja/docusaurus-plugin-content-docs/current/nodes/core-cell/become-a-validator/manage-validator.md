# バリデータ情報の管理

オンボーディングが完了したら、バリデータの情報を更新することができます。

バリデーター搭載ホーム](/img/nodes/become-a-validator/image31.png)

バリデータ管理者としてログインし、**ホーム** メニューに移動すると、搭載されているバリデータが表示される。

この画面で、\*\*[Manage Validator]\*\*をクリックすると、バリデータの管理画面に入ります。

バリデータの管理画面](/img/nodes/become-a-validator/image32.png)

ここでは以下の操作が可能です。

1. **Transfer Manager** - Validator Manager アカウントを変更します。

2. **Request Offboarding** - バリデータを削除するためにオフボーディングリクエストを送信する。

3. **杭打ち契約の追加** - **補助杭打ち契約**を追加します。 ステーキング資金を分けるなどの目的で、複数のステーキング契約（\***マルチステーキング契約**）を運用することができます。 この機能は、**パーミッションレス・フェーズ2**ではサポートされないことに注意してください。

4. **Remove Staking Contract** - 補助ステーキング契約を削除する。 これはバリデータに登録されている複数のステーキングコントラクトの1つを削除する。 オンボーディングに使用されたメインのステーキング契約（**コンセンサス・ステーキング契約**）は削除できません。
