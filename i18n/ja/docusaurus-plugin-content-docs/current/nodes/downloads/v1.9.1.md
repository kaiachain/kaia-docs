## リリースノート

このバージョンは、セキュリティパッチとトレーサー機能強化のためのものです。

### 新機能

- ブロック範囲内のトランザクションをトレースする新しいAPIを実装 (#1586 #1590)
  - `debug_traceChain` (`debug_subscribe` with `traceChain` オプション) は、指定した範囲のトレース結果を提供するサブスクリプション ID を返します。 このAPIは、ウェブソケット接続を介して呼び出されなければならない。
  - `debug_traceBlockByNumberRange` は指定したブロック番号の範囲の全てのトランザクションをトレースする。
- yamlを使用した新しいノード設定方法を導入しました。

### 改善点

- chaindatafetcherの範囲フェッチにメモリ制御ロジックを導入 (#1583)
- txトレーサのリファクタリングとデバッグAPIにおけるメモリ制御の強化 (#1581)

### 修正

- 送信者が十分なKLAYを持っていない場合に発生するestimateGas APIの失敗を解決しました。
- 特別に細工されたp2pメッセージによるノードのパニックを防止 (https://github.com/klaytn/klaytn/pull/1592/commits/f7270496cb11f248e78d261f289fa4e473da961d)

### その他

- 軽整備（#1566 #1576 #1584 #1585 #1589 #1591）
