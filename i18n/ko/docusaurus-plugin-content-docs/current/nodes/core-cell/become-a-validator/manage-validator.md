# 유효성 검사기 정보 관리

온보딩이 완료되면 유효성 검사자 정보를 업데이트할 수 있습니다.

![온보드 유효성 검사기가 있는 홈](/img/nodes/become-a-validator/image31.png)

유효성 검사기 관리자로 로그인하고 **홈** 메뉴로 이동하면 온보딩된 유효성 검사기를 확인할 수 있습니다.

이 화면에서 \*\*[검증자 관리]\*\*를 클릭하여 검증자 관리 화면으로 들어갑니다.

![유효성 검사기 관리 화면](/img/nodes/become-a-validator/image32.png)

여기에서 다음 작업을 수행할 수 있습니다.

1. **전송 관리자** - 유효성 검사기 관리자 계정을 변경합니다.

2. **오프보딩 요청** - 오프보딩 요청을 제출하여 유효성 검사기를 제거합니다.

3. **스테이킹 계약 추가** - **보조 스테이킹 계약**을 추가합니다. 스테이킹 자금을 분리하는 등의 목적으로 여러 개의 스테이킹 계약(**멀티 스테이킹 계약**)을 운영할 수 있습니다. 이 기능은 무허가 2단계에서는 \*\*지원되지 않으므로 특별한 사유가 없는 한 사용하지 않는 것을 **강력히 권장**합니다.

4. **스테이킹 계약 제거** - 보조 스테이킹 계약을 제거합니다. 이렇게 하면 유효성 검사기에 등록된 여러 스테이킹 컨트랙트 중 하나가 제거됩니다. 온보딩에 사용된 메인 스테이킹 컨트랙트(**컨센서스 스테이킹 컨트랙트**)는 삭제할 수 없으며, 나중에 추가된 보조 스테이킹 컨트랙트만 삭제할 수 있다는 점에 유의하시기 바랍니다.
