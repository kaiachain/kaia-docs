# 밸리데이터 정보 변경

온보딩이 완료된 후에는 밸리데이터의 정보를 변경할 수 있습니다.

![온보딩된 밸리데이터가 표시된 Home](/img/nodes/become-a-validator/image31.png)

Validator manager로 로그인 후 Home 메뉴로 이동하면 온보딩된 밸리데이터를 확인할 수 있습니다.

이 화면에서 \[Manage Validator\] 버튼을 클릭하여 밸리데이터 관리 화면으로 진입할 수 있습니다.

![Manage Validator 화면](/img/nodes/become-a-validator/image32.png)

여기서는 다음과 같은 동작을 수행할 수 있습니다.

1. **Transfer Manager** - Validator Manager 계정을 변경할 수 있습니다.

2. **Request Offboarding** - 밸리데이터로 오프보딩을 신청합니다.

3. **Add Staking Contract** - 보조 스테이킹 컨트랙트(auxillary staking contract) 추가 기능입니다. 스테이킹 자금 분리 등을 목적으로 여러 개의 스테이킹 컨트랙트 (multi staking contracts) 를 운용할 수 있습니다. 단, 이 기능은 Permissionless Phase 2 에서는 지원되지 않을 예정이므로, 특별한 사유가 없는 경우 사용하지 않는 것을 강력하게 권장합니다.

4. **Remove Staking Contract** - 보조 스테이킹 컨트랙트 삭제 기능입니다. 밸리데이터에 등록된 여러 개의 스테이킹 컨트랙트 중 하나를 삭제합니다. 단, 온보딩에 사용했던 주 스테이킹 컨트랙트(consensus staking contract)는 삭제할 수 없으며, 나중에 추가한 보조 스테이킹 컨트랙트(auxillary staking contract)만 삭제할 수 있습니다.
