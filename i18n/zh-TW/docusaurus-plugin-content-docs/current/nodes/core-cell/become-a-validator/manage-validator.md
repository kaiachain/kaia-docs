# 管理驗證器資訊

上線完成後，您可以更新驗證員資訊。

![內建驗證器的首頁](/img/nodes/become-a-validator/image31.png)

以驗證器管理員身份登入並導覽至 **Home** 功能表後，您就可以看到內建的驗證器。

在此畫面中，點選 **[Manage Validator]**，進入驗證器管理畫面。

![管理驗證器畫面](/img/nodes/become-a-validator/image32.png)

您可以在此執行下列動作。

1. \*\* 轉移管理員\*\* - 變更驗證器管理員帳號。

2. \*\* 請求離職\*\* - 提交離職請求以移除驗證器。

3. **新增定價合約** - 新增**輔助定價合約**。 您可以操作多重注冊合約（**多重注冊合約**），以達到分隔注冊資金等目的。 請注意，無權限第二階段**將不支援此功能，因此我們**強烈建議不要使用此功能\*\*，除非您有特別的理由。

4. **移除定價合約** - 移除輔助定價合約。 這會移除註冊到驗證器的多個定價合約中的一個。 請注意，不能移除上線時使用的主要定價合約（**共識定價合約**）；只能移除後續新增的輔助定價合約。
